ChatGPT Collapsible Turns

Collapse and expand conversation turns for a cleaner, faster ChatGPT experience.

v1.2.3

Features

  • Collapsible message rows — click any summary to expand or collapse a turn
  • DOM virtualization — collapsed content is physically removed from the DOM, not just hidden, for real memory and render savings
  • Auto-collapse — keeps only the last 10 messages expanded automatically
  • Bulk actions — collapse or expand all messages from the toolbar
  • Light & dark mode — syncs with ChatGPT's own theme switcher
  • Persistent state — collapse preferences are remembered during your session
  • No data collection — everything stays local in your browser

Installation

Available on the Chrome Web Store — one click to install, no sign-in required.

Changelog

v1.2.3 Latest March 2026
  • Updated selector from article to section[data-turn] for ChatGPT's March 2026 DOM changes
  • DOM virtualization: collapsed turns are physically detached into a DocumentFragment, cutting memory usage and eliminating render work for hidden content
  • Light & dark mode via CSS custom properties, synced to ChatGPT's html.dark / html.light class as well as system preference
  • Role icons (robot for assistant, person for user) in summary rows
  • Stable turn ID caching prevents expand/collapse loop when message IDs shift
  • Observer self-mutation guard stops the extension's own DOM changes from re-triggering the observer
  • Rescan interval increased to 10 s and deferred via requestIdleCallback to eliminate periodic main-thread jank
v1.2.2 Bug fix November 2025
  • Fixed extension showing "ON" at boot while the toolbar was not appearing
v1.2.1 Minor November 2025
  • Minor changes
v1.2.0 November 2025
  • First public release
  • Collapsible turns with preview text and role badge
  • Auto-collapse: keeps last 10 messages expanded
  • Bulk collapse / expand all from the top toolbar
  • Persistent collapse state via sessionStorage
  • Throttled MutationObserver to reduce CPU usage during streaming
